# QM-UX
qm-ux is a JavaScript library for QIANMI admin web UI.

### Features
 * 对JS中常见的一些对象进行了拓展
 

<br/>


### Usage

```js
import QMUX from 'qm-ux';
import {Form,Input,Select} from QMUX; 
render(){
    return(
        <Form>
            <Item><Input/></Item>
            <Item><Select/></Item>
        </Form>
    )
}
```

You can use any element for the list and its elements, not just `ul`/`li`.
